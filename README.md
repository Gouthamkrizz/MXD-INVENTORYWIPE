# MXD-INVENTORYWIPE
A lightweight FiveM inventory wipe script with full QBCore and ESX support. Designed to work with most popular inventory systems including qb-inventory, ox_inventory, qs-inventory, and more. Admin-only, configurable, and easy to install. Ideal for server wipes, testing, punishments, and economy resets. Clean, optimized, and plug-and-play.
